# Metric Definitions



## Revenue

Sum of item price times quantity, aggregated to the user-week level. The analysis uses simulated revenue with an 8% uplift applied to treated high-LTV users in the post period.



## Orders / user

Count of distinct orders per user in a given week.



## AOV (Average Order Value)

Revenue / Orders (defined only when Orders > 0).



## Revenue per user (primary KPI)

Total revenue per user during the observed post-period. Weekly revenue per user is also used for panel-based robustness checks such as DiD.



## Business impact

Cumulative revenue lift per treated user minus an estimated 10% discount cost. This is a simplified revenue-cost calculation, not a full profit or margin model.
