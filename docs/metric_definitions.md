\# Metric Definitions



\## Revenue

Sum of (item\_price × quantity) at the order level, aggregated to the user-week level.



\## Orders / user

Count of distinct orders per user in a given week.



\## AOV (Average Order Value)

Revenue / Orders (defined only when Orders > 0).



\## Revenue per user (primary KPI)

Total revenue per user during the post-period (and also computed weekly for DiD).
